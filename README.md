# ODYSSEY
DBMS final project.<br>
A web application to help you plan your road trip

<img src="public/images/app.png">

## features
1. Add stops along the way
2. Track expenditure
3. Add notes to stops

## stack
Minimal with no bloat.
<ul>
    <li>HTML, CSS, JS, JQuery</li>
    <li>NodeJS</li>
    <li>PostgreSQL</li>
</ul>

## build
Clone the repo and start the server using node. The backend currently only supports postgres, hence you will need
a postgres server running.
